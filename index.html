SHC 2.0

Raspberry Pi SmartHome Control 2.0

View project on GitHub

Das Raspberry Pi SmartHome Control ist eine Heimautomatisierungs Software für den Raspberry Pi.

Funktionen

  • steuern von Funksteckdosen (von diversen Herstellern)
  • steuern von GPIOs
  • Wake On Lan Funtkion
  • abfrage wer zu Hause ist
  • umfangreiche Zeitsteuerung
  • abfragen von Verschiedenen Umweltsensoren (auch in Vebindung mit Arduinos)
  • Ereignisse von Sensoren oder wenn jemand nach Hause kommt oder geht
  • Bedingtes Schalten z.B.: Temperatur im Raum muss min. 20°C sein oder jemand muss zu Hause sein
  • Möglichkeit alles dezentral zu installieren
